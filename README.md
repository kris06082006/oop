## 🛠 Технические требования и стандарты
* **Стандарт:** C++20 / C++23.
* **Архитектура:** Строгое следование паттернам **MVP**.
* **Принципы:** Соблюдение **SOLID** и **DRY**, использование умных указателей.
* **Качество кода:** Обязательный статический анализ и проверка на утечки памяти с помощью **Valgrind**.
* **Тестирование:** Unit-тестирование логики **Catch2** с покрытием методов >90%.
* **Документация:** Полное описание публичных интерфейсов в формате **Doxygen**.

## 📊 Особенности реализации
* **UML:** Проектирование архитектуры начинается с диаграмм классов.
* **Multithreading:** Параллельные вычисления с использованием примитивов синхронизации и анализом производительности.
* **Containers:** Реализация собственных шаблонных контейнеров и итераторов (без использования STL-контейнеров в учебных целях).
* **UI:** Интерактивный диалоговый интерфейс с корректной обработкой ошибок и исключений.
